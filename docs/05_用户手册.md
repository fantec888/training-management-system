# 用户手册

## 启动方式

后端：

```powershell
cd src/backend
.\mvnw.cmd spring-boot:run
```

前端：

```powershell
cd src/frontend
npm install
npm run dev -- --host 127.0.0.1
```

访问地址：`http://127.0.0.1:5173`

## 登录账号

- 管理员：`admin / 123456`
- 物业客服：`service_manager / 123456`
- 财务：`finance_admin / 123456`

## 功能说明

- 首页仪表盘：查看运营指标、待办事项和业务概览。
- 小区管理：维护小区基础档案。
- 住户管理：维护住户档案、认证状态和启停状态。
- 楼栋房屋：维护楼栋、房屋和入住状态。
- 报修工单：处理报修派单、进度流转和删除。
- 投诉建议：提交投诉建议，客服进行处理回复。
- 收费管理：维护账单、费用项目和缴费记录。
- 停车管理：查看车辆和车位使用情况。
- 公告活动：发布、下架和删除公告活动。
- 智能服务：查看公告通知，维护社区活动，登记快递代收并标记领取。
- 数据统计：查看户数、入住率、报修、缴费率和投诉处理时效。
- 安防管理：维护门禁设备在线状态，查看车辆信息，新增巡检任务并填写完成结果。
- 系统用户：维护用户、角色、菜单权限、按钮权限和授权关系。

## 权限说明

- 管理员拥有全部模块和按钮权限。
- 物业客服可维护智能服务、住户、报修、投诉、公告等运营数据。
- 财务账号主要维护收费管理，并可查看数据统计。
- 工程账号可处理报修、房屋和安防巡检相关数据。
