# 测试报告

## 测试环境

- JDK：17
- 后端：Spring Boot、Maven、MyBatis、MyBatis-Plus、PageHelper、H2/MySQL 兼容配置
- 前端：Vue 3、Vite、Element Plus
- 数据库：本地 H2，保留 MySQL profile 配置

## 接口联调结果

- `/api/smart-services`：可返回公告通知、社区活动、快递代收和汇总指标。
- `/api/statistics/advanced`：可返回户数、入住率、报修、缴费率和投诉处理统计。
- `/api/security`：可返回门禁设备、车辆信息、巡检任务和安防汇总指标。
