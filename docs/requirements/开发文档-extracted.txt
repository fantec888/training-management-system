一、项目背景与目标
 
1.1 项目简介
 
智慧物业管理系统是一个基于 Spring Boot + Vue3 的物业管理平台，用于提升小区物业管理效率和服务
质量。
1.2 技术栈
 
后端：
Spring Boot
前端：
Vue 3
1.3 实训目标
 
在8个半天内完成一个可运行的智慧物业管理系统，包括：
✅ 完整的用户权限管理
✅ 基础数据管理（小区、楼栋、房屋、业主）
✅ 物业服务功能（报修、投诉）
✅ 财务管理（账单、缴费）
✅ 数据统计展示
✅ 前后端完整对接

4.1 代码规范
 
后端规范：
 
1. 包结构：
2. 命名规范：
类名：大驼峰（PascalCase），如 SysUserInfoController
方法名：小驼峰（camelCase），如 getUserList
变量名：小驼峰，如 userName

包名：全小写，如 com.example.spms.controller

4.2 数据库规范
 
1. 表命名： t_模块名 或 sys_模块名，如 t_user_info 、sys_role_info
2. 字段命名： 小写+下划线，如 user_name 、create_time
3. 必备字段：
4. 索引规范：
主键自动创建索引
外键字段创建索引
频繁查询字段创建索引
联合查询创建联合索引
5. 数据类型：
字符串：VARCHAR(长度适中，不超过255用VARCHAR，超过用TEXT)
整数：INT、BIGINT（根据范围选择）
小数：DECIMAL(10,2)（金额相关）
日期：DATETIME（不用TIMESTAMP）
布尔：TINYINT(0/1)
